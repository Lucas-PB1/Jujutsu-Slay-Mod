# Documentacao do Projeto

Esta pasta concentra guias de implementacao, assets e planejamento do mod.

## Arquivos

- `card-guide.md`: padrao para criar, revisar e registrar cartas.
- `asset-guide.md`: medidas, pastas e nomes esperados para assets.
- `asset-handbook.md`: workflow visual, decisoes praticas e criterios de aceite.
- `asset-adjustments.md`: log de ajustes feitos ou pendentes em assets especificos.
- `jjk-card-checklist.md`: backlog de cartas e sistemas planejados.
- `jjk-full-reference.md`: referencia criativa de Jujutsu Kaisen para novas ideias.

## Como usar

Para implementar uma carta nova:

1. Leia `card-guide.md`.
2. Consulte `jjk-card-checklist.md` para status e prioridade.
3. Use `asset-guide.md` para gerar as artes no tamanho correto.
4. Registre ajustes visuais relevantes em `asset-adjustments.md`.

Para trabalhar em assets:

1. Leia `asset-guide.md` para medidas e caminhos.
2. Leia `asset-handbook.md` para estilo e workflow.
3. Atualize `asset-adjustments.md` quando fizer uma correcao relevante.

Para brainstorm de conteudo:

1. Use `jjk-full-reference.md` como fonte criativa.
2. Transforme ideias aprovadas em itens no `jjk-card-checklist.md`.
3. So marque um item como concluido quando codigo, localizacao, asset e registro estiverem prontos.

## Regra de manutencao

- Mantenha os arquivos em texto simples, sem emojis e sem caracteres corrompidos.
- Prefira secoes curtas, listas objetivas e caminhos exatos.
- Quando um padrao mudar, atualize primeiro o guia principal e depois os logs/checklists afetados.
